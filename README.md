# Oleksandr Atlassian Portfolio

A personal portfolio website for an Atlassian consultant, built with Next.js and featuring the "Daylight Terminal" design system.

## Design System: Daylight Terminal

A light-themed terminal aesthetic that combines the familiar visual language of code editors with clean, accessible design:

- **Typography**: JetBrains Mono for code/headings, IBM Plex Sans for body text
- **Color palette**: Syntax-highlighting inspired colors (teal, orange, purple, green, red, blue)
- **Components**: Terminal windows, blinking cursors, typewriter effects, command prompts

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS with custom terminal theme
- **Fonts**: JetBrains Mono + IBM Plex Sans (with Cyrillic support)
- **Language**: TypeScript
- **i18n**: English and Ukrainian (EN/UK)

## Getting Started

```bash
npm install      # Install dependencies
npm run dev      # Start development server (localhost:3000)
npm run build    # Build for production
npm run start    # Start production server
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Deployment

Deployed on [Vercel](https://vercel.com). Push to `main` branch triggers automatic deployment.
